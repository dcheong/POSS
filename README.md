How to install map tiles:
  Utilizing Android Studio, access the terminal and enter the following commands:
    adb shell rm -r /sdcard/osmdroid/
    adb shell mkdir /sdcard/osmdroid/
    adb push MAPQUEST.zip /sdcard/osmdroid
